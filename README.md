# Scapia Developer Skills Marketplace

A Claude Code plugin marketplace from Scapia providing skills to enhance developer productivity and code quality workflows.

## Installation

```
/plugin marketplace add Arun-Krishnan-Scapia/security-skill-test
```

## Available Plugins

| Plugin | Description |
|---|---|
| `git-workflow` | Smart git branch management, PR summaries, and conflict resolution |
| `code-health` | Code quality analysis with complexity metrics and refactor suggestions |
| `api-tester` | Generate REST API test suites from OpenAPI specs or curl examples |
| `env-sync` | Synchronize environment configs and credentials across development machines |
